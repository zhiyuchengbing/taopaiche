# CLIP模型微调策略说明

## 🎯 只微调最后几层

系统已配置为**只微调最后几层**，这是一个高效的迁移学习策略。

### 📋 配置参数

在 `config.py` 中设置：

```python
# 微调配置
FREEZE_BACKBONE = True      # 是否冻结主干网络
NUM_LAYERS_TO_FINETUNE = 3  # 微调的层数（从后往前数）
```

### 🎨 三种微调模式

#### 模式1: 只微调最后3层（推荐）⭐
```python
FREEZE_BACKBONE = True
NUM_LAYERS_TO_FINETUNE = 3
```

**优点：**
- ✅ 训练速度快（2-3倍）
- ✅ 显存占用少（可以增大batch size）
- ✅ 不容易过拟合
- ✅ 保留CLIP的预训练特征

**适用场景：**
- 数据集较小（< 10000张图片）
- 显存有限（6-8GB）
- 快速实验和调参

**预期效果：** 准确率 80-90%

---

#### 模式2: 微调最后6层
```python
FREEZE_BACKBONE = True
NUM_LAYERS_TO_FINETUNE = 6
```

**优点：**
- ✅ 更灵活，能学习更多任务特定特征
- ✅ 效果通常比模式1好
- ⚠️ 需要更多显存

**适用场景：**
- 中等数据集（10000-50000张）
- 显存充足（8-12GB）
- 追求更好的效果

**预期效果：** 准确率 85-95%

---

#### 模式3: 微调所有层
```python
FREEZE_BACKBONE = False
```

**优点：**
- ✅ 效果可能最好
- ⚠️ 训练慢
- ⚠️ 容易过拟合
- ⚠️ 需要大量数据

**适用场景：**
- 大数据集（> 50000张）
- 显存充足（12GB+）
- 有充足的训练时间
- 任务与CLIP预训练数据差异大

**预期效果：** 准确率 90-98%（如果数据充足）

### 🔍 工作原理

#### ViT模型（如ViT-B/32）

```
CLIP ViT-B/32 结构:
├── Patch Embedding      [冻结 ❄️]
├── Transformer Block 1  [冻结 ❄️]
├── Transformer Block 2  [冻结 ❄️]
├── ...
├── Transformer Block 9  [冻结 ❄️]
├── Transformer Block 10 [训练 🔥] ← 倒数第3层
├── Transformer Block 11 [训练 🔥] ← 倒数第2层
├── Transformer Block 12 [训练 🔥] ← 倒数第1层
└── Projection Layer     [训练 🔥]
```

**NUM_LAYERS_TO_FINETUNE = 3** 表示只训练最后3个Transformer Block和投影层。

#### ResNet模型（如RN50）

```
CLIP ResNet50 结构:
├── Conv1               [冻结 ❄️]
├── Layer1 (残差块)      [冻结 ❄️]
├── Layer2 (残差块)      [冻结 ❄️]
├── Layer3 (残差块)      [冻结 ❄️]
├── Layer4 (残差块)      [训练 🔥]
└── Attention Pool      [训练 🔥]
```

对于ResNet，只训练最后一个残差块（Layer4）和注意力池化层。

### 🔧 调优建议

#### 1. 从少到多
先用少量层数训练，观察效果，再逐步增加：
- 3层 → 6层 → 9层 → 所有层

#### 2. 根据数据量调整
- **小数据集（< 5000张）**: 3层
- **中数据集（5000-20000张）**: 3-6层
- **大数据集（> 20000张）**: 6层或更多

#### 3. 显存优化
如果显存不足：
```python
FREEZE_BACKBONE = True
NUM_LAYERS_TO_FINETUNE = 3
BATCH_SIZE = 16  # 减小batch size
```

#### 4. 学习率调整
微调最后几层时，可以使用稍大的学习率：
```python
LEARNING_RATE = 1e-5  # 比全量微调(1e-6)大一些
```

---

### 📈 效果监控

训练时观察：

1. **训练速度**: 冻结层数越多，速度越快
2. **损失下降**: 应该稳定下降
3. **验证准确率**: 应该达到85%以上
4. **可训练参数**: 建议在5-20%之间

如果验证准确率不理想：
- 增加 `NUM_LAYERS_TO_FINETUNE`
- 增加 `NUM_EPOCHS`
- 调整 `LEARNING_RATE`

---

### ⚠️ 注意事项

1. **保存的模型**: 无论训练时冻结哪些层，保存的模型包含所有参数
2. **推理速度**: 冻结层数不影响推理速度
3. **模型大小**: 保存的模型大小相同（都是完整的CLIP模型）
4. **兼容性**: 训练好的模型可以正常用于预测和评估
